When reporting bugs, please include:

  • Plakar version and operating system
  • Steps to reproduce the issue
  • Error messages and logs
  • Expected vs. actual behavior

Security Issues & CVEs

Found a security vulnerability? Please report it responsibly.

Important: Do not disclose security vulnerabilities publicly on GitHub or community forums.

We take security seriously at Plakar. If you discover a vulnerability, please contact us directly. We'll work with you to address it promptly.

Report via Email

security@plakar.io

Include in your report:

  • Description of the vulnerability
  • Steps to reproduce
  • Potential impact
  • Suggested remediation

Our Commitment

  • Initial response within 24 hours
  • Progress updates during resolution
  • Public disclosure with your consent
