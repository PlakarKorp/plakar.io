// Your new data protection journey

Backup, Duplicate, Check, Restore, Monitor
all in one smooth flow.

Data backup is no longer a project — it's a seamless action that brings you peace of mind.

Quick start
$ export PLAKAR_PASSPHRASE=-HelloWorld-  # Keep your passphrase safe
$ plakar at ~/backups create
 
$ plakar at ~/backups backup /etc
backup: created 003affd1 of size 9.8 MB in 262.229062ms
 
$ plakar at ~/backups ls
003affd1 9.8 MB 0s /etc
$ export PLAKAR_PASSPHRASE=-HelloWorld-  # Keep your passphrase safe
$ plakar at sftp://nestor@nas.plakar.io/var/backups create
 
$ plakar at ~/backups sync to sftp://nestor@nas.plakar.io/var/backups
sync: synchronized 1 snapshot
info: sync: synchronization completed: 1 snapshots synchronized
 
$ plakar at sftp://nestor@nas.plakar.io/var/backups ls
2025-04-08T10:14:37Z 003affd1 9.8 MB 0s /etc
$ export PLAKAR_PASSPHRASE=-HelloWorld-  # Keep your passphrase safe
$
$ plakar at ~/backups check --fast # Structural check
check: verification of 003affd1:/etc completed successfully
 
$ plakar at ~/backups check # data integrity check
check: verification of 003affd1:/etc completed successfully
$ # Everything is good our data is safe
$ export PLAKAR_PASSPHRASE=-HelloWorld-  # Keep your passphrase safe
$ # Let's start the UI to search and restore only one file
$ plakar at ~/backups ui
 
$ # Or we can fully restore a specific backup
$ plakar at ~/backups ls # Let's find the right backup
003affd1 9.8 MB 0s /etc
$ plakar at ~/backups restore -to /tmp/backups 003affd1
restore: restoration of 003affd1:/etc at /tmp/backups completed successfully
