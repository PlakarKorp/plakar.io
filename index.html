// Your new data protection journey

Backup, Duplicate, Check, Restore, Monitor
all in one smooth flow.

Data backup is no longer a project — it's a seamless action that brings you peace of mind.

Quick start
plakar:~ export PLAKAR_PASSPHRASE=-HelloWorld-  # Keep your passphrase safe
plakar:~ plakar at ~/backups create
plakar:~ plakar at ~/backups backup /etc
2025-04-08T10:12:37Z info: backup: created unsigned snapshot 003affd1 of size 9.8 MB in 262.229062ms
plakar:~ plakar at ~/backups ls
