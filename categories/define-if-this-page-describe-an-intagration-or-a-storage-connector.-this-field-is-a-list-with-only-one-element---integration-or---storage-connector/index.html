How Plakar integrations
and storage connectors work?

Integrations are bundling all the necessary components to back up, restore and visualize your data source.

Storage Connectors are the components that allow you to store, retrieve and duplicate your Kloset.

Kloset is our immutable data engine that turns backups into portable, encrypted, queryable snapshots.
