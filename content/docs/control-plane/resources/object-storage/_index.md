Object storage resources represent bucket-based storage systems that store data as objects rather than files or blocks. They are commonly used as backup stores due to their durability, scalability, and low cost.

Plakar Control Plane supports the following object storages:

- [GCS]()
- [S3](./s3)
- [Azure Blob]()
