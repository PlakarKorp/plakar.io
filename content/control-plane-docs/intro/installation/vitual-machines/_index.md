# Virtual Machines

Plakar Control Plane can be deployed as a virtual machine on any hypervisor or
cloud platform that supports virtual machine images. Depending on your
environment, Plakar Control Plane is distributed in several formats such as
`OVA`, `VMDK` and `ISO`.

Each guide in this section covers the deployment process for a specific platform
or environment, including how to import the appropriate image format and
configure the virtual machine before first boot.
